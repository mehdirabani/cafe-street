# ☕ Cafe Street - Landing Page فروشگاهی

> تمرین پیاده‌سازی یک صفحه فرود فروشگاهی با استفاده از **Flexbox**

این پروژه بر اساس طرحی از جامعه Figma با نام [Cafe Street - E-Commerce Landing Page](https://www.figma.com/design/NAnJcD6qQWkmfk4VXCvqCD/Cafe-Street---E-Commerce-Landing-Page--Community-?node-id=205-256&t=lLT9j2GnggvcAqFV-0) طراحی شده است. هدف اصلی، تمرین و تسلط بر قابلیت‌های **CSS Flexbox** برای چیدمان المان‌ها در صفحه است.

## 🎯 اهداف پروژه

- پیاده‌سازی pixel-perfect یا نزدیک به طرح اصلی
- استفاده از Flexbox برای تمام بخش‌های چیدمان (navbar، محصولات، فوتر و ...)
- بهبود مهارت در ساختاردهی HTML و CSS خالص (بدون فریم‌ورک)
- ایجاد یک صفحه واکنش‌گرا (Responsive) با کمک Flexbox

## 🛠 تکنولوژی‌ها

- HTML5
- CSS3 (با تمرکز بر Flexbox)
- Font Awesome یا آیکون‌های دلخواه (اختیاری)
- گوگل فونت (اختیاری)

## 🚀 راه‌اندازی و اجرا

1. مخزن را کلون کنید:
```bash
git clone https://github.com/mehdirabani/cafe-street.git
```
2. وارد پوشه پروژه شوید:
```bash
cd cafe-street
```
3. فایل index.html را در مرورگر باز کنید (با دوبار کلیک یا استفاده از Live Server در VS Code).
